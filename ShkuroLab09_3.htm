Лабораторна робота №9 — Використання різних класів

Вступ Основна частина Висновки
Розділ 1 — Вступ

Це перший розділ з основним текстом, оформленим за допомогою класу main-text. Клас можна використовувати багато разів у документі. CSS класи дозволяють застосовувати однакове форматування до різних елементів, що значно спрощує підтримку веб-сайту.

Виноска: тут можна додати додаткову інформацію про розділ або посилання на джерела.

Розділ 2 — Основна частина

Основна частина містить кілька абзаців з однаковим форматуванням. Використання класу main-text забезпечує єдність стилю. Це дуже зручно, коли потрібно змінити оформлення всіх текстових блоків одночасно.

Виноска: додаткові пояснення до розділу. Виноски зазвичай містять другорядну, але корисну інформацію.

Розділ 3 — Висновки

У висновках підводимо підсумки попередніх розділів. Текст повторно використовує клас main-text. CSS класи є потужним інструментом для створення стильного та послідовного дизайну веб-сторінок.

Виноска: приклад повторного використання класу footnote для демонстрації універсальності підходу.
